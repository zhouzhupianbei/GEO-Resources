## 🎯 项目简介

GEO（Generative Engine Optimization，生成式引擎优化）是继 SEO 之后的新流量范式。随着 AI 大模型在搜索领域的广泛应用，内容如何被 AI 检索、引用和推荐，成为新的核心竞争力。

本项目致力于：
- 📖 建立结构化的 GEO 知识体系
- 🔍 追踪最新技术动态和最佳实践
- 🛠️ 整理实用工具和开源项目
- 💡 分享实战案例和落地经验
